AI assistant

Bring your own API key to recommend, organize, and write intros.

Enable AI

  1. 1

    Open AI assistance

    Turn on “AI Assistance” under “Settings → Metadata & Lyrics → AI Assistance”.

  2. 2

    Pick a provider

    Under “Connection”, choose a provider (OpenAI, Anthropic, Gemini, Zhipu, DeepSeek and 40+ presets, or a custom OpenAI-compatible endpoint) and enter the Base URL.

  3. 3

    Enter API key and model

    Enter your API key, then pick an instant model and a reasoning model (type a custom name or choose from “Available Models”).

  4. 4

    Test the connection

    Tap “Test Connection” to verify, or use “AI Trial Chat” to send a message and confirm the AI is ready.

Dual model roles

TuneSync distinguishes two model roles, auto-selected per task:

Instant modelFast and low-cost — AI DJ recommendations, filename cleanup, etc.
Reasoning modelDeep and high-quality — album intros and complex tasks; reasoning effort is configurable

AI DJ

The AI DJ builds a play queue from your listening profile, with three autonomy levels:

  • Preview before play: surface recommendations for review
  • One-tap play: play immediately after confirmation
  • Automatic continuation: auto-add songs as the queue runs low

Tool calling

With tool calling enabled, the model can use these tools (some permission-gated):

  • web_search: DuckDuckGo web search
  • fetch_webpage: fetch webpage content
  • search_album_art: search and upscale artwork (100×100 → 1200×1200)
  • search_lyrics: search lyrics
  • search_library: search within your local library
  • control_playback: control playback (requires permission)
  • manage_playlist: manage playlists (requires permission)
  • Custom tools: tools you define

About privacy

AI calls use your own API key and go directly to your chosen provider. Enable tool calling and “smart tool selection” as needed — keep capability while controlling cost.

Trial chat

Under “AI Assistance” there’s a trial chat to verify your connection works without entering a specific feature.
